# Mobil Uygulama Akışları ve Gereksinimler

## Amaç
Web ile aynı backend'i kullanan, Android ve iOS uyumlu mobil uygulama.

## Temel Akışlar
- Kullanıcı: Kayıt, giriş, hizmet arama, randevu alma, mesajlaşma, ödeme, randevu takibi
- Sağlayıcı: Kayıt, profil oluşturma, hizmet ekleme, randevu yönetimi, mesajlaşma, ödeme takibi

## Özel Gereksinimler
- Expo ile hızlı build
- Push notification desteği (Expo Notifications)
- Kamera ve galeri erişimi (profil fotoğrafı, belge yükleme)
- Mobilde sadeleştirilmiş UI/UX

## Ekranlar
- /login, /register, /home, /search, /provider/:id, /booking, /messages, /profile, /payments
