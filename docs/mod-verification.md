# Sağlayıcı (Mod) Doğrulama Süreci

## Akış Adımları
1. Sağlayıcı kayıt olur ve profilini doldurur.
2. Gerekli belgeleri (kimlik, diploma vb.) yükler.
3. Admin, başvuruyu inceler.
4. Onaylanırsa sağlayıcı aktif olur ve hizmet sunabilir.
5. Reddedilirse, eksikler bildirilir ve tekrar başvuru yapılabilir.

## Akış Diyagramı (metin)
- Sağlayıcı → Profil ve belge yükleme → Admin incelemesi → (Onay) → Aktif
- Sağlayıcı → Profil ve belge yükleme → Admin incelemesi → (Red) → Eksik bildirimi → Tekrar başvuru
