# Test Stratejisi ve Senaryoları

## Amaç
Sistemin güvenli, hatasız ve sürdürülebilir olmasını sağlamak.

## Test Türleri
- Birim Testleri (Unit Test)
- Entegrasyon Testleri
- API Testleri
- UI/E2E Testleri

## Kullanılacak Araçlar
- Backend: Jest, Supertest
- Frontend: React Testing Library, Cypress

## Örnek Senaryolar
- Kullanıcı kayıt/giriş işlemleri doğru çalışıyor mu?
- Sağlayıcı doğrulama süreci hatasız işliyor mu?
- Randevu oluşturma ve ödeme akışı sorunsuz mu?
- Admin panelinde kullanıcı/sunum yönetimi doğru mu?
